**№1 Ответ:** `1`

**Объяснение:** При первом выполнении цикла (когда `$i = 1`) происходит сложение `$sum = 0 + 1` и сразу же выполняется `return $sum`, что завершает функцию. Цикл не продолжается.

**Что хотел сделать автор:** Вероятно, автор хотел вычислить сумму чисел от 1 до `$num`.

**Исправленный код:**
```php
<?php
function func($num) {
    $sum = 0;
    for ($i = 1; $i <= $num; $i++) {
        $sum += $i;
    }
    return $sum;
}
echo func(5);
?>
```
Этот код выведет `15`.
