**№1** - `5+++` (условие истинно, выполняется первая команда после if, затем выполняется вторая команда всегда)

**№2** - `+++` (условие ложно, первая команда после if не выполняется, вторая команда выполняется всегда)

**№3** - `---+++` (условие ложно, первая команда после if не выполняется, вторая и третья команды выполняются всегда)
