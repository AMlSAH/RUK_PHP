**№1 Ответ:**  
На экран будет выведено:  
`12`

**Объяснение:**  
- `$num1 = 1`  
- `$num2 = $num1` — `$num2` получает копию значения `1`  
- `$num2 = 2` — `$num2` изменяется на `2`, `$num1` остаётся `1`  
- `echo $num1` выводит `1`  
- `echo $num2` выводит `2`  
Результат: `1` и `2` выводятся слитно.

- `$num2++` увеличивает значение по ссылке на `1` → `$num1` и `$num2` становятся равны `2`  
- `echo $num1` выводит `2`  
- `echo $num2` выводит `2`  
Результат: дважды `2`.
