**№1 Результат:**  
Ошибка уровня `Notice: Undefined variable: aaa` и ничего не выведется (или пустой вывод, в зависимости от настроек). Переменная `$aaa` не определена внутри функции, так как имеет глобальную область видимости.

**№2 Результат:**  
Ошибка уровня `Notice: Undefined variable: num1` (и аналогично для `num2`). Функция `func()` не имеет доступа к глобальным переменным `$num1` и `$num2` без явного указания (например, через `global`). Возвращаемое значение не определено, и `echo` выведет ошибку.
