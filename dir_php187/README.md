**№1 Ответ:**  
На экран будет выведено:  
`1!`

**Объяснение:**  
- `$arr1 = [1, 2, 3, 4, 5]` — создаётся массив  
- `$arr2 = $arr1` — `$arr2` получает копию массива  
- `$arr2[0] = '!'` — изменяется только `$arr2[0]`, `$arr1[0]` остаётся `1`  
- `echo $arr1[0]` выводит `1`  
- `echo $arr2[0]` выводит `!`  
Результат: `1` и `!` выводятся слитно.

**№2 Ответ:**  
На экран будет выведено:  
`22`

**Объяснение:**  
- `$arr1 = [1, 2, 3, 4, 5]`  
- `$arr2 = &$arr1` — `$arr2` становится ссылкой на `$arr1`  
- `$arr1[0]++` увеличивает первый элемент → `$arr1[0]` и `$arr2[0]` становятся `2`  
- `echo $arr1[0]` выводит `2`  
- `echo $arr2[0]` выводит `2`  
Результат: дважды `2`.
